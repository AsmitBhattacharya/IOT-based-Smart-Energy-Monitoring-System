🛠️ System OverviewThe IoT-Based Energy Management System provides a complete end-to-end solution for real-time residential power monitoring. By combining precision hardware sensors with a localized web dashboard, the system allows users to track consumption metrics and improve energy efficiency.🔌 Hardware ImplementationThe hardware setup utilizes an ESP32 microcontroller as the central processing unit, interfaced with a PZEM-004T sensor module.Real-time Sensing: The PZEM-004T measures AC voltage, current, active power, and energy consumption with high accuracy.Local Display: An integrated LCD/OLED module provides an immediate physical readout of current metrics (e.g., 231.0V and 0.04A) for on-site monitoring.Prototypes & Iteration: The project transitioned from breadboard prototyping to a functional MVP, ensuring robust hardware-to-software communication.📱 "Energy Console" Web InterfaceThe system hosts a responsive web dashboard accessible via any device on the local network.Real-time Analytics: Displays live data for Voltage (V), Current (A), Power (W), and Energy (Wh).Mobile Optimized: Designed with a clean, "Dark Mode" UI for easy reading on smartphones.Edge Inference Ready: Built to support local edge inference for appliance state classification using TinyML.
